# WiFi Jammer Detection System (Simulation)

A simple Python simulation that detects possible WiFi jamming attempts by identifying high deauthentication packet counts.

## Features
- Detects suspicious WiFi activity
- Logs high deauth packet counts
- Simulates real cybersecurity IoT behavior

## Usage
```
python wifi_jammer_detection.py
```
